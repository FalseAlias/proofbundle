# ProofBundle

ProofBundle is a local browser verifier for sealed JSON evidence bundles.

Current public status: `public_alpha`.

Release-green status is not claimed.

## Use

Open the app:

- GitHub Pages: https://falsealias.github.io/proofbundle/
- Direct verifier: https://falsealias.github.io/proofbundle/web/proofbundle_minimal_verifier_20260517.html

The verifier runs in the browser. Files stay local.

## Public Boundary

This public repository surface is intentionally small. It should not publish
private material, internal coordination records, private corpus material, or
withdrawn material.

The current public-surface record is:

- `PUBLIC_BOUNDARY_SUPERSESSION_20260517.json`

## Development

The implementation is static HTML and JavaScript. No build step is required for
the public verifier page.

# Public Minimal

ProofBridge is a hash-ratcheted feedback substrate for persistent multi-agent state.

Instead of relying on an AI agent to remember, ProofBridge externalizes continuity into a verified loop. Each agent output becomes a predecessor-bound bridge record; the verifier checks lawful succession; the append-only ledger head becomes the next stimulus for another agent.

The agents may be transient. The loop persists.

Core phrase:

> Persistence is not inside the agent. Persistence is in the verified loop.
